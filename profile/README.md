## 🚀 How to Start - 어케 시작하는지 적기
### Clone Repository 
```
$ git clone --recursive https://github.com/A-Interview/Docker.git <- 우리꺼로 해야함
```
### env setting in the Settings folder 
* settings/.env
```
# backend env
django_SECRET_KEY=

MYSQL_DATABASE=
MYSQL_NAME=
MYSQL_USER=
MYSQL_PASSWORD=
MYSQL_ROOT_PASSWORD=
MYSQL_PORT=
MYSQL_HOST=

# GPT & DALL-E
GPT_KEY=

# AWS env
Access_key_ID =
Secret_access_key =
AWS_BUCKET_NAME = 
FILE_URL=

# Naver
JIN_YONG_NAVER = 
Naver_client_ID =
Naver_client_SECRET = 

```
### Run Docker
```
$docker compose up --build
```
